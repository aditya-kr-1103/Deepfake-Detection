# Model Weights

Place the DFDC-trained EfficientNet-B7 checkpoint here as:

```text
backend/weights/dfdc_effnet_b7.pt
```

Alternatively set:

```powershell
$env:DEEPFAKE_MODEL_PATH="C:\path\to\checkpoint.pt"
```

The PDF describes the architecture and runtime pipeline, but it does not contain trained model weights.
